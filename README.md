### getting started
You need [gulp-cli](https://github.com/gulpjs/gulp-cli) installed globally. I'm using Node version >= 16, but other versions will likely work.

## build
```sh
yarn build
```
or
```sh
npm run build
```
## run locally
```sh
yarn start
```
or
```sh
npm start
```
